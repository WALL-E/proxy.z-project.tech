SwitchySharp

SwitchySharp 是 Google Chrome 浏览器上的一个代理管理扩展程序，目前最好用的代理程序。

配置

如果您使用 goagent 或 shadowsocks 代理，可以在 SwitchySharp “选项”的“导入/导出”中可选择下载 SwitchyOptions.bak 备份文件后从文件恢复或使用 http://switchysharp.com/file/SwitchyOptions.bak 地址来在线恢复备份（如下图所示），恢复后可以看到已经配置好 goagent 和 shadowsocks 代理，另外您也可以选择手动配置相关代理。

配置完代理之后需要启动启动相应的代理程序才可以正常使用代理，可以查看代理使用。
