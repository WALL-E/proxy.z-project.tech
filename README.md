# 目标
本项目需要完成的目标

* 统一入口
  * 失败自动重试
  * 自动切换访问代理
* 支持多种代理
  * http
  * https
  * sock5
* 灵活的付费方式
  * 免费
    * 限制访问次数
    * 限制访问流量
  * 付费
    * 按照流量付费
    * 按照请求次数付费
* IP资源
  * 网络
    * 定时爬取
    * 定时清洗
  * 众筹 
    * ADSL用户(拥有外网IP)
    * 机房用户(拥有外网IP)
    * NAT用户(需要隧道协议)
